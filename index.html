What is Python?

Python is an interpreted, high-level and general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.[27] Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly, procedural), object-oriented, and functional programming. Python is often described as a "batteries included" language due to its comprehensive standard library.

Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. A function can return data as a result.

Listas

Acceder
tu_lista[0] - Accesar al primer dato de la lista, de la misma manera puedes modificarlo, se puede combinar con otras opciones tu_lista[0].title()
tu_lista[-1] - Accesar el ultimo elemento de una lista. Puedes acceder al penultimo por ejemplo agregando [-2]
Agregar
tu_lista.append("elemento") - Agregando elementos al ultimo lugar en la lista.
tu_lista.insert( 1, "elemento") - Insertando un elemento en un sitio en especifico.
Eliminar
tu_lista.pop() - Elimina el ultimo elemento de una lista, si agregas un numero va a eliminar el elemento en es index que le indiques pop(0) , también te permite guardar ese ultimo elemento en una variable. var_reservar = tu_lista.pop()
tu_lista.remove('nombre_elemento') - Para remover un elemento en especifico.
del tu_lista[2] - Elimina un elemento en el index que le indiques.
Acomodar
tu_lista.reverse() - Para cambiar el orden en una lista permanente.
tu_lista.sort() - Organiza una lista de forma permanente, si agregas reverse=True organisa la lista al reves.
tu_lista.sorted() - Organiza una lista de forma temporal
len.(tu_lista) - Encontrar el tamaño de una lista.
Cortar pedazos de la lista
tu_lista[0:3] - Para cortar una lista.
[:3] - Para cortar desde el principio.
[2:] - Para cortar desde un index dado hasta el final.
[-2:] - Para cortar los últimos elementos del la lista.
[2:10, 3] - Para cortar del elemento 2 al 9 en saltos de 3.
Copiar
tu_lista[:] - Para copiar una lista.
